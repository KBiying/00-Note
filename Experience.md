# 如何看项目源码

看github上的项目，首先要学会用，要把这个项目用起来，跑起来，然后再去看代码，很多录友都反过来了，项目都没运行都没跑起来，然后就开始专研代码，那是绝对看不懂的。

所以看开源项目可以分这么几步：

1. 运行，跑起来，用起来，知道这个项目有什么功能，满足什么需求

2. 找到项目入口 main函数

3. 拆解项目模块，都有哪些功能，哪些模块

4. 一个模块一个模块去看代码，而不是 囫囵吞枣

5. 修改部分代码，重新跑项目，看看有哪些变动

6. 完成这个项目代码的阅读
